W niniejszym wpisie poruszę temat tworzenia pliku .gitignore krok po kroku w pustym repozytorium.
Na codzień pracuję na systemie operacyjnym Windows.

Na samym początku otwieramy konsolę (ja używam cmder http://cmder.net/) - wyszukujemy w niej miejsca gdzie znajduje się nasz projekt.

Kolejne kroki wyglądają następująco:

  • git init
  • touch .gitignore
  • otwieramy przeglądarkę internetową i wpisujemy https://www.gitignore.io/

W pasku wyszukiwania zaznaczamy VisualStudio i naciskamy przycisk Create.
Na powstałym pliku wykonujemy Ctrl + A oraz Ctrl + C
Plik powinien wyglądać w następujący sposób:

  • code .gitignore (w moim przypadku otwiera się Visual Studio Code)
    Można użyć do tego celu innego edytora tekstu np. Sublime Text, Notatnika, itd.
    Do pliku .gitignore należy wkleić oraz zapisać powstały wcześniej plik.
    Ctrl + V oraz Ctrl + S
    W tym momencie można już zamknąć Visual Studio Code.

  • git status

  • git add . lub git add -A lub git add –all (ja osobiście używam git add -A)
  • git status
  • git commit -m “Initial commit”
  • git remote add origin https://github.com/NazwaUżytkownika/NazwaRepozutorum.git
  • git push
  • podaj nazwę użytkownika oraz hasło
